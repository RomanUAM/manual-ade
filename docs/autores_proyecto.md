## Regla editorial

Estos nombres se usan para identificar autorias y colaboraciones academicas del ecosistema de trabajo. El sistema no debe atribuir a estas personas contenidos, resultados o permisos de publicacion que no esten verificados en el material fuente.

Cuando un material sea ajeno o no tenga permiso explicito, puede alimentar la base interna de conocimiento, pero no debe publicarse ni descargarse desde la pagina.
